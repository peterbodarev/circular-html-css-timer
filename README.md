## How To Use 🔧

Example of HTML code:

```html
<div class="timer">
	<div class="timer__line"></div>
	<div class="timer__body">
		<div class="timer__counter">
			<span>1</span>
			<span>2</span>
			<span>3</span>
			<span>4</span>
			<span>5</span>
			<span>6</span>
			<span>7</span>
			<span>8</span>
			<span>9</span>
			<span>10</span>
		</div>
	</div>
</div>
```
